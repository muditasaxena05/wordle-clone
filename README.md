# wordle-clone

A simple Wordle clone I built to practice frontend development. Runs entirely in the browser, it has no frameworks, no dependencies, just one HTML file.

##How to play:
Guess the 5-letter word in 6 tries. After each guess the tiles tell you how close you were:

- Green tile: Right letter, right spot
- Orange tile: Right letter, wrong spot
- Grey tile: Letter isn't in the word

Type with your keyboard or use the on-screen keys.

##Features:
- 500+ word list
- Flip and shake animations
- Win streak tracker
- Works on mobile too
